# FlowGen Architect

A robust, client-side only architecture diagramming tool for creating isometric system designs and visual flows.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build for production:
   `npm run build`
